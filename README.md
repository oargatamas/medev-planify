# Medev Planify

## Description

Medev Planify is a tour organizing platform that connects users with partners, offers in-app chat, and tracks the planning progress, simplifying the entire trip organization process.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
